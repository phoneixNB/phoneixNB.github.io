CSS基础

发表于 2021-08-26 更新于 2021-08-27

CSS:

层叠样式表，用来美化网页。做到结构（html）和表现（CSS）分离

基本语法：

选择器{

属性：属性值；

}

CSS引用方式：行间样式，内部样式，外部样式，导入外部样式

行间样式：直接在标签上书写样式

内部样式：在文件的内部书写样式

外部样式：（1）先创建一个CSS文件（2）再用link标签引入这个文件

导入外部样式：（1）先创建一个CSS文件；（2）在style标签中用import导入这

​ 个样式文件。

以上四种区别：行间用于当前标签：内部用于当前文件：外部样式可以被多个HTMl引用。在实际项目开发中，最好使用外部样式。外部样式分为，link引入和import引用。我们更多的使用link

CSS选择器分类：

（1)*：匹配html中所有元素，*号的性能非常差，因为他要匹所有元素，所以不建议使用

（2）：标签选择器：用来匹配对应的标签

（3）：类选择器：是用来选择匹配class命名的标签

（4）：id选择器：用来选择用id命名的标签

（5）：派出选择器，根据上下文来确定选择的标签

（6）：伪类选择器

CSS选择器分组：让多个选择器（元素）具有相同的样式，一般用于设置公共样式。

CSS选择器继承：子元素可以继承父元素的样式，反之不成。

CSS优先级：外部样式<内部样式<嵌入样式

样式权重：!important(10000)>内联样式（1000）>id选择器（100）>类和伪类（10）>标签（1）

CSS字体：

font-size:字号

font-family:字体

font-style:文字样式：normal:默认值，italic：斜体，oblique：倾斜的字体

font-weight:文字加粗：normal：正常，bold：加粗,bolder：比bold粗，lighter，比normal细，{100-900},400=normal,700=bold

line-height:行高

color:文字颜色

text-decoration:文字修饰,nomal:默认值，underline：下划线，line-through:贯穿线；overline：上划线

text-align:文本对齐方式

text-transform:字母大小写：capitalize：将每一个单词的第一个字母转化成大写，uppercase：转化成大写，lowercase：转化成小写

text-indent:设置文本首行文本缩进

font-variant：normal/small-caps(让大写字母小一些)

tip：

font复合属性

​ font：font-style font-variant font-weight font-size/line-height font-family

（1）注意属性值的位置顺序

(2) 除了font-size和font-family，其他属性值都可以省略

CSS背景：

1）background-color：背景色(transparent/color)

2）background-image：背景图（none/url）

3）background-repeat：背景图像铺排方式

（repeat/no-repeat/repeat-x/repeat-y）

4）background-position：设置对象的背景图像位置

(x-number,top,center,bottom)

(y-number,left,cencter,right)

5）background-attachment：背景图像滚动位置scroll/fixed

6）background：设置背景的复合写法：

background：color image repeat attachment position

CSS伪类选择器

伪类：专门用来表示元素的一种特殊状态。

常用的伪类选择器：

​ a标签的伪类：

​ :link/:visited/:hover/:active

​ :focus获得焦点

​ :first-child/:last-child/:nth-child(number)

属性选择器

[属性名]：包含有指定属性名的元素（常用）

[属性名=值]：属性名的值为指定值的元素（常用）

[属性名~=值]：属性名的值包含指定值的元素

[属性名^=值]：属性名的值以指定值的开头的元素

[属性名$=值]：属性名的值为指定值的结尾的元素

关系选择器

1）空格：后代选择器

2）>：只选择儿子

3）+：兄弟选择器

CSS伪元素

:before/:after/:fist-letter/:first-line:前面可以是一个冒号，也可以两个

::selection/::placeholder/::backdrop:前面只能是双冒号

content用于before和after中的文本插入。

html学习笔记1

发表于 2021-08-25

标签属性

通用属性：除单标签外都有的属性

id;用来给标签取一个唯一的名称

class;用来给标签取一个类名

style;用来设置该标签的行列样式

title；当鼠标移到该标签上所显示提示内容

自定义标签属性:通常用来传值或用于图片懒加载

格式：data-*

表格标签：table，主要用于呈现格式化数据。表格是由行和列组成。

格式：

table的属性

border：设置表格边框

width：设置表格宽度

align；设置表格对齐，left,right,center。指的是表格。不是表格内容

cellpadding；设置单元格间距

cellspacing；设置像素间隙

现在的table属性可能更改过，用ws虽然可以使用但是显示弃用

th表头有加粗居中的作用

td的属性：

rowspan=“合并的数目”：用来合并上下单元格

colspan=“合并的数目”：用来合并左右单元格

align：内容居中

valign=“top/bottom/middle”：垂直居中(顶端/底部/中间)

如果想对一整行操作，直接在tr里写

table的完整表格：

caption：标题

thead：表头

tbody：表体

tfoot：表尾

form表单标签：用来实现前后端交互的一个重要标签

常用属性：

name：表单名称

action：表单数据提交到哪里去（通常是后台文件名）。如果是#，表示提交到当前文件

method：前端提交数据到后端的方法，主要有：get和post。默认的是get

get提交数据会暴露在地址栏，post不会

表单元素：

input类：用来完成输入，选择或发出指令

type值有：

text；单行输入文本框，是一个默认值。

常用属性有：placeholder（提示）/name(命名)/ min/maxlength(最少/最多输入的字符个数)/disabled(失效)/readonly(只读)/value(用来设置默认值)/pattern(正则匹配)

password；密码框

常用属性有：与text相同

radio：单选钮，通常是两项以上

常用属性有：name（必须要有）/value/checked(默认选中)/disabled/readonly

checkbox：复选框，检查框。用来选择0，1，多项。属性与radio相同

file；文件上传按钮，没有什么属性

button：普通按钮，通常用来调用脚本代码

常用属性有：value（按钮的标题）/disabled（失效）

image；图片按钮，用法与button一样，有一个特殊属性，src（用来加载提示图片，用它替换了value），ta有提交功能，与submit一样。

submit；提交按钮，用来将表单数据提交到后台

常用属性有：value（按钮的标题）/disable（失效）

reset；重置按钮，将表单所有组件输入的内容全部清空，还原为初始

常用属性有：value（按钮的标题）/disable（失效）

textarea类：文本域（多行文本框），主要用于输入大量内容、

常用属性：

name/id/cols（列数）/rows（行数）

/placeholder/minlength/maxlength/reqrired(必须输入)/value

select类:下拉列表框，用于单项选择。用option实现每一个选项。

常用属性：multiple：用于多选，此时变成列表框

​ size:最多显示的行数

button类：普通按钮，具有提交功能，可以单独使用，不屑在form中；如果写在form中就有提交功能

iframe：框架集，是用来将多个网页文件组合成一个文件

常用属性：

name：框架名

src：引入外部的html文件

scrolling：滚动条（yes/no/auto）

width:宽度

heigth：高度

frameborder：是否有边框（1/0）

marginheight：框架离顶部和底部的距离（%/px）

marginwidth:框架离左边和右边的距离（%/px）

注意在实际开发中尽量减少iframe，因为它破坏前进和后退功能，且不利于Seo。可以使用div来做。
